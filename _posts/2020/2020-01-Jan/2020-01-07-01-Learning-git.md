## Learning git

If you want to publish a post, these are tehe steps to follow :

1. Clone the repo
2. git init
2. Make a branch (kavya)
3. Add a new post
4. `git add -A`
5. `git commit -m "message"`
6. `git push origin kavya`
7. Login to github.com and make a Pull Request.
